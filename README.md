# anicode

Find unicode characters matching your search string

## Install

With [Fisherman]

```
fisher i anicode
```

## Usage

```
fisher anicode [options] [--quiet] [--help]
```

## Options

* -q, --quiet<br>
    Enable quiet mode.

* -h, --help<br>
    Show usage help.
